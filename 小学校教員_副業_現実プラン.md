# 小学校教員のための「現実的に続けられる」副業プラン

## 小学校教員ならではの3つの制約

1. **平日は時間がない**：朝7時出勤〜夜8時退勤、持ち帰り仕事も多い。月の残業は平均60〜80時間。
2. **児童・保護者にバレるリスクが高い**：地域に住んでいる保護者の目があるため、対面型・露出型は危険。
3. **同僚・管理職の目**：職員室の文化的に「副業＝悪」のイメージがまだ強い学校も多い。

ただし、教員には **有利な制度的特例** があります。

> ### 🎓 教育公務員特例法 第17条
> 「教育公務員は、教育に関する他の職を兼ね、又は教育に関する他の事業若しくは事務に従事することができる」
>
> → **「教育に関する」副業は、一般公務員より許可が下りやすい**
> （所属長＝校長ではなく **任命権者＝教育委員会** の許可が必要）

## 🏆 小学校教員に最も現実的な副業ランキング

### 第1位：投資（新NISA）— ★★★★★ 圧倒的におすすめ

| 項目 | 内容 |
| --- | --- |
| 必要時間 | **月10分**（積立設定だけ） |
| バレるリスク | **ゼロ**（許可不要、確定申告不要のケース多い） |
| 児童・保護者リスク | **ゼロ** |
| 想定収入 | 元本次第。長期で年4〜6% |

**具体策**：
- 新NISA積立投資枠で **eMAXIS Slim 全世界株式（オール・カントリー）** を月3〜5万円積立
- 成長投資枠で **高配当ETF（VYM、SCHD、1489）** を購入
- iDeCoも併用（教員は月1.2万円まで・所得控除あり）

**到達イメージ**：
- 月5万円×20年・年利5%＝**2,055万円**
- 高配当ETFに置き換えれば **年80〜100万円の配当**

⚠️ FX・暗号資産・信用取引・個別株デイトレは **依存性＋損失リスク** から非推奨。

### 第2位：執筆（教材・教育書・記事監修）— ★★★★★ 教員の特権

教員の本業と直結し、**教育公務員特例法17条で許可が下りやすい** ジャンル。

| 副業内容 | 報酬目安 | 許可 |
| --- | --- | --- |
| 教育系出版社のワークブック・ドリル執筆 | 1冊 5〜30万円 | 要 |
| 教育雑誌の連載・記事執筆（『教育技術』『総合教育技術』『教職研修』など） | 1本 5,000〜3万円 | 要 |
| 指導書・赤刷り教科書の執筆協力 | 1冊 10〜50万円 | 要 |
| 教員向け書籍の単著・共著 | 印税10%×定価×部数 | 要 |
| Web教育メディアの記事監修（みんなの教育技術、EduAなど） | 1本 5,000〜2万円 | 要 |

**現実的な目標**：
- 雑誌記事 月1本 × 1.5万円 ＝ 年18万円
- 共著書を年1冊 ＝ 10〜30万円
- **合計：年30〜50万円**

**始め方**：
1. 教育雑誌の編集部に「実践報告」を投稿（無料寄稿でまず実績作り）
2. 採用されたら担当編集に「有償執筆も可能か」打診
3. 教育委員会に **兼業許可申請** を出す（許可が下りやすいジャンル）

### 第3位：講演・研修講師 — ★★★★☆

| 副業内容 | 報酬目安 |
| --- | --- |
| 教育委員会主催の研修講師 | 1回 1〜3万円 |
| 民間教育研究会（東京書籍・明治図書等のセミナー） | 1回 3〜10万円 |
| 大学の教職課程 非常勤講師（土曜開講） | 月3〜8万円 |
| 教員志望者向け予備校（東京アカデミー等）の単発講座 | 1回 1〜3万円 |

⚠️ **平日夜・土日に活動できる人向け**。
⚠️ 教育委員会の許可必須。

### 第4位：不動産投資（管理委託） — ★★★☆☆

時間がない教員にも向いている。ただし **初期費用が必要**。

- 親から相続したアパート・実家の賃貸経営
- 区分マンション1〜2室の購入
- 条件：**4棟9室未満／年500万円未満／管理会社に委託** → 許可不要

ローンを組んで始める場合、教員は属性が良く銀行ローンが通りやすいのが利点。
ただし「**ワンルーム投資の電話勧誘**」はカモにされるので絶対に契約しないこと。

## 💡 教員の長期休暇を活用した副業

夏休み（30〜40日）・冬休み（10日）・春休み（10日）は教員の特権です。

- **教材制作の集中作業**（執筆・編集）
- **書籍の原稿執筆**
- **オンライン教員コミュニティでの講師活動**（事前許可必須）
- **教育系YouTubeの台本制作（顔出しなしで配偶者・知人名義の運営に協力）** ※ただしグレー

## ⛔ 小学校教員が「やってはいけない」副業

| 副業 | 理由 |
| --- | --- |
| ❌ **塾講師・家庭教師（対面）** | 同業競合。教育公務員特例法でも生徒との利害関係問題で許可されにくい |
| ❌ **YouTube収益化（顔出し）** | 児童・保護者に確実にバレる。炎上事例多数 |
| ❌ **Uber Eats・配達業** | 同じ校区で配達中に保護者と遭遇 → 苦情・処分事例あり |
| ❌ **アフィリエイト事業** | 事業性ありで許可必要。教育系ブログは生徒情報の流用リスク |
| ❌ **メルカリの転売** | 仕入れ転売は事業。古物商許可も必要 |
| ❌ **保護者・PTA関係者との金銭取引** | 利害関係 → 信用失墜 |
| ❌ **個別指導塾の運営協力** | 同業の利益相反 |

## 🗓️ 始めるロードマップ（教員向け）

### 1ヶ月目：投資の基盤を作る
- [ ] 証券口座（SBI証券 or 楽天証券）を開設
- [ ] 新NISAで全世界株式インデックスを月3万円積立スタート
- [ ] iDeCoの加入手続き（教員は月1.2万円上限）

### 3ヶ月目：執筆実績を作る
- [ ] 『教育技術』『総合教育技術』に **無料投稿** から始める
- [ ] 教育系ブログ（個人）で実践記録を蓄積（収益化はしない）
- [ ] 教員コミュニティ（X、Facebook、note）で発信

### 6ヶ月目：許可申請を出す
- [ ] 出版社から原稿依頼が来たら **教育委員会に兼業許可申請**
- [ ] 校長に事前相談（味方につける）
- [ ] 確定申告の準備（freee・マネーフォワード）

### 1年目：副収入を安定化
- [ ] 投資配当・執筆料・講演料の合計を年30〜50万円に
- [ ] 翌年は単著・連載に挑戦して年100万円超を目指す

## ✅ 最重要チェックリスト

- [ ] 副業時間が **本業の準備・成績処理を圧迫しない**
- [ ] **児童の個人情報・学校の内部情報** は絶対に使わない・書かない
- [ ] **校区内の保護者**との金銭取引は避ける
- [ ] 副業所得 **年20万円超は確定申告**、住民税は **「普通徴収」** を選択
- [ ] SNS・ブログでは **学校名・地域名・児童の特定情報** を出さない
- [ ] **教育委員会の兼業規程** を毎年確認（自治体により細部が違う）

## まとめ：教員に最適な「黄金の組み合わせ」

> ### 🥇 **新NISA積立投資（時間ゼロ）＋ 教育系執筆・講演（許可制・本業に活きる）**
>
> これが小学校教員にとって
> - **時間効率最高**
> - **バレるリスク最小**
> - **本業のスキルにも還元される**
> - **教育公務員特例法で守られている**
>
> ベストな組み合わせです。
>
> 年120万円はいきなりは難しいですが、
> **3〜5年で年60万円、10年で年120万円** は十分現実的な目標です。
