# 毎朝5時起き・5km走れるようになるプラン

## プロフィール
- 年齢: 44歳 男性
- 身長/体重: 168cm / 68kg（BMI 24.1 → 標準範囲）
- 運動歴: 大学まで部活でサッカー（基礎体力・運動神経あり）

---

## フェーズ概要（全12週）

| フェーズ | 期間 | 目標 |
|--------|------|------|
| Phase 1 | 1〜3週目 | 早起き習慣の定着・歩行メインで体を慣らす |
| Phase 2 | 4〜6週目 | ウォーク＆ランを交互に（インターバル走） |
| Phase 3 | 7〜9週目 | ジョギング中心・3km連続走を目指す |
| Phase 4 | 10〜12週目 | 5km完走・習慣として定着 |

---

## Phase 1（1〜3週目）：起床習慣と体の慣らし

### 目的
- 5時起きのリズムを体に刻む
- 膝・腰・アキレス腱に負荷をかけず関節を慣らす
- 大学卒業後のブランクを考慮し、焦らない

### 週スケジュール（月〜土）
| 曜日 | 内容 | 時間目安 |
|-----|------|--------|
| 月 | 起床後ストレッチ＋ウォーキング30分 | 約35分 |
| 火 | 休息（軽いストレッチのみ） | 10分 |
| 水 | ウォーキング30分 | 30分 |
| 木 | 休息 | — |
| 金 | ウォーキング30〜40分 | 40分 |
| 土 | ウォーキング40分（ペースアップ） | 40分 |
| 日 | 完全休息 | — |

### 就寝時間の目標
- 23時00分就寝 → 5時00分起床（6時間睡眠）
- 最初の1週目は23時30分、2週目は23時15分と段階的に早める

### ポイント
- アラームは1つだけ（スヌーズ禁止）
- 起きたら水を一杯飲み、外の空気を吸う
- 走る準備（シューズ・ウェア）を前夜に必ず用意

## 早起きを継続するための戦略

### 睡眠環境
- 寝室は暗く・涼しく（18〜20度が理想）
- スマホは寝る1時間前から見ない
- 就寝前のアルコールは控える（睡眠の質を下げる）

### 朝のルーティン（5:00〜5:45）
```
5:00  起床 → 水を飲む
5:05  動的ストレッチ（5分）
5:10  着替えてシューズを履く
5:15  走り始め
5:45  帰宅 → シャワー or 汗を拭く
```

### モチベーション維持
- ランニングログをつける（距離・時間・天気・体調）
- Stravaなどのアプリ活用
- 月1回、近所の5kmコースを走りタイムを記録

## ストレッチ・ケアメニュー（毎回必須）

### 走る前（動的ストレッチ）5分
1. 足首回し（各10回）
2. 膝の屈伸（10回）
3. ランジウォーク（10歩）
4. 股関節ほぐし（前後スイング 各10回）
5. その場かけ足（30秒）

### 走った後（静的ストレッチ）10分
1. ふくらはぎ（各30秒）
2. 太もも前（大腿四頭筋）（各30秒）
3. ハムストリングス（各30秒）
4. 股関節（鳩のポーズ）（各30秒）
5. アキレス腱（各30秒）

## 足首の痛みへの多角的対処法

> サッカー経験者は足首の古傷・慢性的な不安定性を抱えていることが多い。
> 「痛みがある状態で走る」は厳禁。以下を段階的に実施する。

### Step 0：まず痛みの種類を把握する

| 痛みのタイプ | 考えられる原因 | 優先対応 |
|------------|-------------|--------|
| 走った後に痛む・翌朝痛い | 炎症・腱への過負荷 | 即休養＋アイシング |
| 歩いても痛む | 捻挫後遺症・疲労骨折の疑い | 整形外科受診 |
| 動かし始めだけ痛い | 関節の硬化・腱鞘炎 | ストレッチ強化＋受診検討 |
| 特定の方向に曲げると痛い | 靭帯の不安定性 | テーピング＋受診 |

**「歩いても痛む」「2週間以上続く」場合は必ず整形外科へ。**

---

### アプローチ 1：急性期の対処（RICE処置）

痛みが出た直後や炎症がある場合：

```
R  Rest（安静）   → 走るのを即中止。ウォーキングも控える
I  Ice（冷却）    → 患部に氷嚢を当て15〜20分。1日2〜3回
C  Compression   → 弾性包帯やサポーターで軽く圧迫
E  Elevation     → 就寝時・安静時に足を心臓より高く上げる
```

アイシングの目安：**運動後30分以内**に行うと効果的。

---

### アプローチ 2：医療・専門家への相談

| 専門家 | 何を相談するか |
|-------|-------------|
| **整形外科** | レントゲン・MRIで骨・靭帯の状態確認。疲労骨折・靭帯損傷の除外診断 |
| **理学療法士（PT）** | 足首の可動域・筋力評価。リハビリプログラム作成 |
| **スポーツ外来** | アスリート向け診療。競技復帰を見越した治療方針 |
| **足専門の靴店（フィッティング）** | インソール・シューズ選定で足首への負荷を軽減 |

---

### アプローチ 3：セルフケア・リハビリ

#### 足首の柔軟性改善（毎日実施）
1. **タオルギャザー**：床に置いたタオルを足指でたぐり寄せる（各足1分）
2. **足首アルファベット**：足首で空中にA〜Zを書く（可動域全方向）
3. **ふくらはぎストレッチ**：壁に手をついて30秒×2セット
4. **アキレス腱ストレッチ**：膝を軽く曲げた状態でも実施（深部まで伸ばす）

#### 足首の安定性強化（痛みが引いてから）
1. **片足立ち**：目を開けて30秒 → 慣れたら目を閉じて30秒
2. **カーフレイズ**：つま先立ちゆっくり上下 10回×3セット
3. **チューブトレーニング**：ゴムチューブで内返し・外返しの抵抗運動
4. **バランスボード**：不安定面でのトレーニング（慣れてから）

---

### アプローチ 4：道具・装備による負荷軽減

| アイテム | 効果 | 目安価格 |
|---------|------|--------|
| **足首サポーター**（走行中着用） | 横ぶれ防止・安定性補完 | 1,500〜3,000円 |
| **カスタムインソール** | 足底のアーチを支え足首への負担を分散 | 3,000〜30,000円 |
| **ヒールカップ付きインソール** | かかとの着地衝撃を吸収 | 2,000〜5,000円 |
| **モーションコントロールシューズ** | 過回内（オーバープロネーション）を制御 | 12,000〜20,000円 |
| **テーピング（非伸縮・キネシオ）** | 走行中の関節固定・運動サポート | 500〜1,500円/本 |

> サッカーでの足首捻挫経験がある場合、**オーバープロネーション（内側への倒れ込み）** が起きやすい。
> シューズ選びは専門店でフォーム分析を受けることを強く推奨。

---

### アプローチ 5：走り方・路面の見直し

| 見直し項目 | 現状の問題 | 改善策 |
|----------|----------|-------|
| **着地方法** | かかと着地は衝撃大 | ミッドフット（足の中央）着地を意識 |
| **歩幅** | 大股走りは足首に負荷集中 | 歩幅を小さく・回転数（ピッチ）を上げる |
| **路面** | アスファルトは硬く衝撃大 | 土・芝のコース、または河川敷を選ぶ |
| **傾斜** | 坂道は足首に側方ストレス | 平坦なコースから始める |
| **走行距離の増やし方** | 急激な増加は故障の元 | 週間距離は10%以上増やさない |

---

### アプローチ 6：足首が痛い期間の代替トレーニング

走れない間も心肺機能・筋力を維持できる：

| 代替運動 | 足首への負荷 | 効果 |
|---------|-----------|------|
| **水中ウォーキング** | 極めて低い | 全身有酸素・関節リハビリ |
| **水泳（クロール）** | ほぼゼロ | 心肺機能維持 |
| **自転車（ロードバイク・エアロバイク）** | 低い | 脚筋力・心肺維持 |
| **上半身トレーニング** | なし | 体幹・上半身筋力維持 |
| **椅子に座ってのストレッチ** | なし | 柔軟性維持・リハビリ |

早起き5時の習慣は「代替運動」で継続しておくと、足首回復後にスムーズに戻れる。

---

### 足首痛がある場合のプラン修正

**痛みがある間はPhase 1を延長・修正する：**

```
通常Phase 1（3週）
    ↓ 足首痛あり
修正Phase 1（4〜6週）：
  ・ウォーキングを痛みのない範囲に限定
  ・アイシング・ストレッチを毎日実施
  ・整形外科受診で原因を特定
  ・許可が出てからPhase 2へ進む
```

**焦りは最大の敵。完治してから走り始めた方が長期的に早く5kmに到達できる。**

## 44歳ランナーへの注意点

| 注意点 | 対処 |
|-------|------|
| 足首の痛み | 上記「足首の痛みへの多角的対処法」を参照 |
| 膝・腰の痛み | 痛みがあれば即休養。無理は禁物 |
| 筋肉痛 | 2〜3日で引かなければ休養延長 |
| 脱水 | 夏場は走る前後に水分補給必須 |
| 心臓への負荷 | 胸の痛み・動悸があれば即中止・受診 |
| 靴 | クッション性の高いランニングシューズを使用（関節保護） |
