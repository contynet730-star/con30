# 毎朝5時起き・5km走れるようになるプラン

## プロフィール
- 年齢: 44歳 男性
- 身長/体重: 168cm / 68kg（BMI 24.1 → 標準範囲）
- 運動歴: 大学まで部活でサッカー（基礎体力・運動神経あり）

---

## フェーズ概要（全12週）

| フェーズ | 期間 | 目標 |
|--------|------|------|
| Phase 1 | 1〜3週目 | 早起き習慣の定着・歩行メインで体を慣らす |
| Phase 2 | 4〜6週目 | ウォーク＆ランを交互に（インターバル走） |
| Phase 3 | 7〜9週目 | ジョギング中心・3km連続走を目指す |
| Phase 4 | 10〜12週目 | 5km完走・習慣として定着 |

---

## Phase 1（1〜3週目）：起床習慣と体の慣らし

### 目的
- 5時起きのリズムを体に刻む
- 膝・腰・アキレス腱に負荷をかけず関節を慣らす
- 大学卒業後のブランクを考慮し、焦らない

### 週スケジュール（月〜土）
| 曜日 | 内容 | 時間目安 |
|-----|------|--------|
| 月 | 起床後ストレッチ＋ウォーキング30分 | 約35分 |
| 火 | 休息（軽いストレッチのみ） | 10分 |
| 水 | ウォーキング30分 | 30分 |
| 木 | 休息 | — |
| 金 | ウォーキング30〜40分 | 40分 |
| 土 | ウォーキング40分（ペースアップ） | 40分 |
| 日 | 完全休息 | — |

### 就寝時間の目標
- 23時00分就寝 → 5時00分起床（6時間睡眠）
- 最初の1週目は23時30分、2週目は23時15分と段階的に早める

### ポイント
- アラームは1つだけ（スヌーズ禁止）
- 起きたら水を一杯飲み、外の空気を吸う
- 走る準備（シューズ・ウェア）を前夜に必ず用意

---

## Phase 2（4〜6週目）：ウォーク＆ランのインターバル

### 目的
- 走る動作を体に再学習させる
- 心肺機能を無理なく引き上げる

### 週スケジュール
| 曜日 | 内容 | 距離目安 |
|-----|------|--------|
| 月 | ウォーク2分→ジョグ1分 ×5セット | 約2km |
| 火 | 休息またはストレッチ | — |
| 水 | ウォーク2分→ジョグ2分 ×5セット | 約2.5km |
| 木 | 休息 | — |
| 金 | ウォーク1分→ジョグ3分 ×4セット | 約2.5km |
| 土 | ジョグ15〜20分（無理しないペース） | 約2km |
| 日 | 完全休息 | — |

### ペース目安
- ジョグ：会話ができるペース（キロ8〜9分）
- 「ゆっくりすぎ」くらいがちょうどいい

### 12週目以降の維持プラン
- 週4〜5回、5km走を継続

---

## 早起きを継続するための戦略

### 睡眠環境
- 寝室は暗く・涼しく（18〜20度が理想）
- スマホは寝る1時間前から見ない
- 就寝前のアルコールは控える（睡眠の質を下げる）

### 朝のルーティン（5:00〜5:45）
```
5:00  起床 → 水を飲む
5:05  動的ストレッチ（5分）
5:10  着替えてシューズを履く
5:15  走り始め
5:45  帰宅 → シャワー or 汗を拭く
```

### モチベーション維持
- ランニングログをつける（距離・時間・天気・体調）
- Stravaなどのアプリ活用
- 月1回、近所の5kmコースを走りタイムを記録

---

## ストレッチ・ケアメニュー（毎回必須）

### 走る前（動的ストレッチ）5分
1. 足首回し（各10回）
2. 膝の屈伸（10回）
3. ランジウォーク（10歩）
4. 股関節ほぐし（前後スイング 各10回）
5. その場かけ足（30秒）

### 走った後（静的ストレッチ）10分
1. ふくらはぎ（各30秒）
2. 太もも前（大腿四頭筋）（各30秒）
3. ハムストリングス（各30秒）
4. 股関節（鳩のポーズ）（各30秒）
5. アキレス腱（各30秒）

---

## 44歳ランナーへの注意点

| 注意点 | 対処 |
|-------|------|
| 膝・腰の痛み | 痛みがあれば即休養。無理は禁物 |
| 筋肉痛 | 2〜3日で引かなければ休養延長 |
| 脱水 | 夏場は走る前後に水分補給必須 |
| 心臓への負荷 | 胸の痛み・動悸があれば即中止・受診 |
| 靴 | クッション性の高いランニングシューズを使用（関節保護） |
