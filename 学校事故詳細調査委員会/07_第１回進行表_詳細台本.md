## 【0分】開会・出席者の紹介（5分間）

「ただいまから、練馬区学校事故詳細調査委員会 第１回を開催いたします。」

「本日は、『重大事態の再発防止策に向けた意見交換』をテーマに、限られた時間ではございますが、皆様から専門的な視点、また学校現場の実情を踏まえたご意見をいただきたいと考えております。」

### ◆ 会議の方針
「実効性のある再発防止策の構築に向けて、建設的で忌憚のない、そして前向きなご意見交換を進めてまいりたいと考えておりますので、どうぞよろしくお願いいたします。」

「〇〇委員」

「〇〇委員」
「教育委員会から、〇〇課長、〇〇課長補佐が出席いただいております。」

### ◆ 事務局から確認

「本日の進行につきましては、事務局からご説明いたします。」

「本日の会議は、全体で60分を予定しており、以下の流れで進行いたします。

- 第1部：開会・出席者の紹介（5分）
- 第2部：事態の概要説明と背景理解（15分）
- 第3部：再発防止策に向けた意見交換（40分）
  ※その中で、課題抽出、具体策の検討、方針の整理を行います
- 第4部：事務連絡・閉会（5分）

また、本日の議事につきましては、議事録として記録いたしますので、ご了承ください。それでは、第2部に移らせていただきます。」

## 【20分～60分】再発防止策に向けた意見交換（40分間）

### ◆ 委員長 進行方法の説明（1分）

「それでは、本日のメインテーマに移りたいと思います。『重大事態の再発防止策に向けた意見交換』です。」

「ここからは、以下の流れで進行いたします：

**第1段階：課題の抽出（12分）**
本事態から浮かび上がってくる課題について、各委員からご意見をいただきます。

**第2段階：具体的な再発防止策の検討（20分）**
学校レベル、教育委員会レベルでの具体的な対策について、ご意見をいただきます。

**第3段階：対応方針の整理（7分）**
本日出た意見をまとめ、今後の方向性を確認いたします。

皆様から、忌憚のないご意見を賜りたいと思いますので、よろしくお願いいたします。」

### ◆ セッション①：課題の抽出（12分間）

**委員長のファシリテーション開始**

「では、まず第1段階として、本事態から浮かび上がってくる課題についてご意見をお伺いしたいと思います。」

「学校現場の視点、教育委員会の視点、保護者・地域の視点など、様々な角度からのご指摘をお願いします。」

「例えば、以下のような点が考えられますが、これに限られるものではございません：

- **事態の早期発見・報告体制の課題**
  事態の現象を見逃さず、迅速に報告する体制が十分か

- **教職員間の情報共有体制の課題**
  学年、分掌、職員全体での情報共有が図られているか

- **保護者・地域との関係構築の課題**

- **対応に必要な知識・スキルの課題**

## 【備考】

### ◆ タイムマネジメント上の注意点

| 項目 | 予定時間 | 実際 | 備考 |
|------|---------|------|------|
| 開会・出席者紹介 | 5分 | | 時間厳守 |
| 概要説明 | 15分 | | 質問は後で |
| 課題抽出 | 12分 | | 全員発言の確保 |
| 具体策検討 | 20分 | | 実現性を重視 |
| 方針整理 | 7分 | | 本日のまとめ |
| 事務連絡・閉会 | 5分 | | 余った時間で深掘りOK |

### ◆ 委員長への指示事項

- 時間厳守：各セッション終了時に時間を確認
- 発言促進：発言が少ない委員には積極的に問いかける
- 対立回避：異なる意見が出た場合は、「いずれも重要な視点」と触れる
- 記録確認：事務局の議事録取りが追いついているか確認
- 雰囲気管理：会議の雰囲気を和やかに、かつ実務的に保つ

### ◆ 投影資料・配布資料

必須資料：
1. 重大事態の概要（投影用スライド）
2. 重大事態対応フローチャート（投影用）
3. 対応方針のまとめ（配布用）

参考資料：
1. 関連する校則・規程
2. 過年度の事案の対応事例
3. 関係法令
